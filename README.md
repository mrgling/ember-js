# Test-app ember.js

# Förkrav

Du behöver detta för att kunna köra applikationen:

* [Node.js](https://nodejs.org/) (with npm)
* [Ember CLI](https://ember-cli.com/)

## Installation

* `npm install`
* `npm install -g ember-cli`

## Starta

* `npm start`
* Gå till: [http://localhost:4200](http://localhost:4200).
